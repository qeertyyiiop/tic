# Tic Tac Toe

To install the dependencies, run:

```bash
npm install
```

To start the application, run:

```bash
npm run start
```

To run the tests, run:

```bash
npm run test
```

To build the application, run:

```bash
npm run build
```

To deploy the application, run:

```bash
npm run build
serve -s dist
```
